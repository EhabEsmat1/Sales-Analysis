# Sales-Analysis
# Sales Analysis Power BI Project

## Overview
This project analyzes sales data using Power BI to extract insights about revenue, customers, and product performance.

## Tools Used
- Power BI
- Excel
- Data Cleaning & Modeling

## Key Insights
- Total Sales performance over time
- Top selling products
- Customer segmentation
- Profit analysis

## Files Included
- .pbix file (Power BI dashboard)
- Excel dataset
- Dashboard screenshots
